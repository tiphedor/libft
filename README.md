# libft
The best C library in the world

# Compilation

    make
  
That's it.

# Usage

Compile with : 

    cc -I path-to-libft/includes -L path-to-libft -lft <...>

Or, alternatively, run : 

    sudo make install
    
To copy the header and lib file into /usr/local. Then compile with :

    cc -lft <...>
